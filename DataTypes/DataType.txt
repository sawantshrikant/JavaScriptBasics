 JavaScript has several built-in data types, which can be categorized into two main groups: 
 primitive data types and reference data types
 1)number 2)string 3 )boolean 4)undefined  5)null 6)symbols

 reference data type

2)Array - Represents an ordered list of values. Elements in an array can be of any data type. 
3)Function - Represents a reusable block of code. Functions can be assigned to variables, passed as arguments, and returned from other functions.
